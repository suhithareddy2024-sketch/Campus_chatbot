# Campus_chatbot
AI-Powered Intelligence for Every Corner of Your Campus.
## Tech Stack
- **Frontend:** Streamlit
- **AI Framework:** LangChain
- **LLM:** Google Gemini Pro
- **Vector Store:** FAISS
- **Maps:** Folium
- **Scraping:** BeautifulSoup
##  Getting Started

### Installation
# Clone the repo
git clone https://github.com/suhithareddy2024/campus-chatbot.git
cd campus-chatbot
### Prerequisites
- Python 3.10+
- Google Gemini API Key
